# Cinephile
Chinephile - 예술 형태로서 영화에 매우 관심이 많고 열정적이며 영화에 대해 많이 알고 있는 사람(영화광)

## 프로젝트 개요
- 영화 후기 작성 및 공유 서비스
- 프로젝트 기간 : 2023.04.26 - 2023.05.08
